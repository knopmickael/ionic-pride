# Ionic 6x - Employees Management

## Project Summary
This is an Ionic project designed to manage employee data, including basic information and profile pictures. The goal of this project is to provide a user-friendly interface for managing employee data and making it easily accessible for a manager.

## Running the Project on the Browser
To run the project on your browser, you can follow these steps:

- Clone this repository by running the following command in your terminal: git clone https://github.com/your-username/your-repo.git
- Navigate to the project directory by running cd your-repo.
- Install the necessary dependencies by running npm install.
- Start the development server by running ionic serve.
- Open your browser and navigate to http://localhost:8100/ to view the project.

That's it! You should now be able to view and interact with the project in your browser.
